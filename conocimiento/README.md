## Cómo agregar nuevo material

1. Convertir el PDF: `./scripts/agregar_pdf.sh <pdf> <destino/>`
2. Revisar el markdown generado
3. Agregar una entrada en este README en la sección correspondiente
4. Ver [GUIA_CONVERSION.md](../GUIA_CONVERSION.md) para más detalles
